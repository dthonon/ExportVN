Procédure d'installation sous Ubuntu 16.04

1) installer git
   $ sudo apt install git

2) installer PHP et les modules requis
   $ sudo apt install php php-pear php-pgsql

3) installer log4php
   $ sudo pear channel-discover pear.apache.org/log4php
   $ sudo pear install log4php/Apache_log4php

4) installer les scripts de téléchargement
   $ git clone https://github.com/dthonon/ExportVN
   
5) configurer une copie locale des scripts de lancement
   $ cd ExportVN
   $ cp ChargePsql.sh ChargePsql_local.sh
   $ cp ExportJson.sh ExportJson_local.sh
   puis éditez ces 2 fichiers pour renseigner les valeurs propres à votre site
